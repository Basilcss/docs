# Pictures

Basil provides a class to make images fit to its container

```css
.pic {
  background-size: cover;
  background-position: center;
  object-fit: cover;
}

@media (min-width: 764px) {
  .big\:pic {
    background-size: cover;
    background-position: center;
    object-fit: cover;
  }
}
```
